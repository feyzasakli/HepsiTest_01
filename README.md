# HepsiTest_01

 Hepsiburada'da ana sayfadan ürün seçip sepete ekleme akışını test ediyorum. Web için Playwright, Android için Maestro kullandım.

## Test Akışı

1. Ana sayfayı aç
2. İlk ürüne tıkla
3. Sepete ekle
4. Login ekranında üye olmadan devam et
5. Sepette ürünün göründüğünü doğrula

## Notlar

Testler Firefox'ta çalışıyor. POM yapısı kullandım, her sayfa kendi sınıfında.
